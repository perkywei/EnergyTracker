# 智慧能耗管家 — 用户使用指南

## 系统简介

车辆能耗记录系统，用于记录加油/充电事件，自动计算油耗、电耗及综合能耗，并以图表形式展示趋势。

## 一、注册

首次使用需要先注册账号。

### 注册步骤

1. 在登录页面点击 **「注册」** 按钮
2. 填写以下信息：
   - **用户名** — 自定义 ID，用于登录
   - **密码** — 登录密码
   - **注册码** — 8 位字母数字码（由管理员提供）
   - **车辆参数**（见下方说明）
3. 阅读弹出的用户协议，勾选 **「我已阅读并同意以上协议」**，点击 **「确认注册」**
4. 注册成功后自动跳转到登录页

### 车辆参数说明

| 参数 | 说明 | 示例 |
|------|------|------|
| 油箱总容量 (升) | 车辆油箱标称容量 | 50 |
| 电池总容量 (kWh) | 动力电池标称容量 | 40 |
| 当前总里程 (km) | 仪表盘当前总里程数 | 15000 |
| 当前HEV里程 (km) | 仪表盘当前 HEV（燃油）里程 | 8000 |
| 当前油量 (%) | 仪表盘当前油量百分比 | 80 |
| 当前电量 (%) | 仪表盘当前电量百分比 | 60 |

> **注意：** HEV 里程不能大于总里程，油量和电量的百分比范围是 0~100。

## 二、登录

1. 输入已注册的用户名和密码
2. 点击 **「登录」**
3. 登录成功后进入主界面，顶部会显示用户名

> **提示：** 登录后刷新页面不会丢失登录状态，会自动恢复。如需退出，点击右上角的 **「退出登录」**。

## 三、记录加油/充电事件

登录后主界面分为三个区域：记录表单、历史记录表格、能耗趋势图表。

### 切换事件类型

点击顶部的 **「⛽ 加油事件」** 或 **「🔋 充电事件」** 切换记录类型。

### 加油事件

切换到加油事件后填写：

| 字段 | 说明 |
|------|------|
| 总里程 (km) | 加油时的仪表盘总里程 |
| HEV里程 (km) | 加油时的仪表盘 HEV 里程 |
| 金额 (元) | 本次加油花费金额 |
| 操作前油量 (%) | 加油前的油量百分比 |
| 操作前电量 (%) | 加油前的电量百分比 |
| 加油后油量 (%) | 加油后的油量百分比（需 ≥ 加油前） |
| 加油量 (L) | 本次实际加油升数 |

### 充电事件

切换到充电事件后填写：

| 字段 | 说明 |
|------|------|
| 总里程 (km) | 充电时的仪表盘总里程 |
| HEV里程 (km) | 充电时的仪表盘 HEV 里程 |
| 金额 (元) | 本次充电花费金额 |
| 操作前油量 (%) | 充电前的油量百分比 |
| 操作前电量 (%) | 充电前的电量百分比 |
| 充电后电量 (%) | 充电后的电量百分比（需 ≥ 充电前） |
| 充电量 (kWh) | 本次实际充电度数 |

填写完成后点击 **「✅ 保存记录」**。

> **提示：** 如果提示"请先设置车辆参数"，点击用户名进入个人中心填写即可。

## 四、查看数据

### 记录明细表

保存后的记录会显示在下方的表格中，内容包括：

- 时间、类型、总里程、HEV里程、EV里程、金额
- 补能量（加油升数或充电度数）
- 区间里程（相邻两次记录之间的行驶里程）
- **油耗** (L/100km)、**电耗** (kWh/100km)、**总能耗** (L/100km)
- 删除按钮（🗑️）

### 能耗趋势图

页面底部的折线图展示油耗、电耗和综合能耗的变化趋势，双 Y 轴显示（左轴：油耗/总能耗，右轴：电耗）。

### 累计花费

表格上方显示 **「累计花费: ¥xxx.xx」**，自动汇总所有记录。

## 五、个人中心

点击主界面右上角的 **用户名**，进入个人中心。

### 车辆参数

可以修改油箱容量、电池容量、当前里程和油量/电量百分比。修改后点击 **「保存车辆参数」**，系统会自动重新计算所有能耗数据。

### 修改密码

切换到 **「🔐 修改密码」** 标签，输入原密码和新密码，点击 **「确认修改」**。修改成功后需要重新登录。

## 六、数据导出与导入

### 导出 Excel

点击 **「📎 导出Excel」** 按钮，将当前所有记录导出为 `.xlsx` 文件。

### 导入 Excel

1. 点击 **「📂 导入Excel」** 按钮
2. 选择之前导出的 `.xlsx` 文件
3. 系统会自动解析并导入记录

> **注意：** 导入时需确保 Excel 格式与导出格式一致。

## 七、清空记录

点击 **「🗑️ 清空所有记录」** 将删除当前账号下的全部能耗记录，操作前会二次确认，请谨慎操作。

## 八、常见问题

### 刷新后需要重新登录？
正常情况下不需要。如果退出登录了，下次打开页面需要重新登录。

### 忘记密码？
联系管理员处理。
